# file-processing-toolkit-cpp
A C++ program that performs file processing operations such as word counting, reversing text, finding the longest word, and searching for words within a file.


This project is a simple C++ program that performs basic text file processing operations. It allows users to analyze and manipulate text files through a menu-driven interface. The program reads data from a file and applies different operations such as counting words, reversing word order, finding the longest word, and searching for specific words.

 Features:

 Word Count – Counts the total number of words in a file

 Reverse Words – Displays all words in reverse order

 Find Longest Word – Identifies the longest word in the file

 Search Word – Checks if a word exists in the file (or how many times it appears)

 File Input Support – Works with user-provided file names

 Technologies Used



Standard Library (<iostream>, <fstream>, <vector>, <string>)

   HOW TO RUN:

Compile the program:

g++ main.cpp -o file_tool
